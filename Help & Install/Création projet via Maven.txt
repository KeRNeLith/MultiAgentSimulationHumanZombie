Dézipper Maven dans un répertoire 
Définir la variable M2_HOME=c:\Program Files\Apache-Maven-x.x.x
Ajouter à la variable PATH : %M2_HOME%\bin
(Attention la variable pour Java doit être définie => JAVA_HOME)

Vérification du bon fonctionnement de Maven:
mvn --version

Exemple résultat:

7+01:00)

Default locale: fr_FR, platform encoding: Cp1252

Commande de génération du projet:
mvn archetype:generate -DgroupId=fr.sma.zombifier -DartifactId=zombifier -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false

Pour compiler le projet:
(Cette commande correspond à une phase du cycle de vie du build)
mvn package

Exécution du programme:
java -cp target/zombifier-0.0.1.jar fr.sma.zombifier.App

Génération du site (Javadoc):
mvn site
